This website documents the development of sparse multiple regression methods for gene set enrichment analysis (GSEA), as well as analysis demonstrating their utility in comparison to typical marginal (one gene set at a time) GSEA techniques. Additionally we provide exposition on GSEA, a review of existing tools for GSEA, and related problems and the ideas motivating the development of these new methods.

The analysis on this website depends on the gseasusie package. If you wan’t to use gseasusie here’s how to get started
